Guestbook Application

View Guestbook

This feature allows you to view all the guestbook entries.

Each record has a full name, email address, guestbook entry message and entry date.

Sign to Guestbook

New entries can be added to the guestbook.

The entry will require a full name, email address and an entry message.

Login

Admin can login with username and password

Admin can approve the entry records from here. Admin can also delete the records from here.
